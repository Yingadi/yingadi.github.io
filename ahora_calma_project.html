A progressive web app designed in collaboration with another developer from DAM course for a new facility to manage users and space reservations. This app allows private individuals to book offices and large rooms for activities and workshops related to health and wellness, such as coaching and yoga.

The app manages users and space reservations. It includes features for different user roles, such as administrators and regular users, each with distinct permissions and capabilities to ensure secure and efficient management of the facility's resources.

My role: I was responsible for designing the HTML pages using CSS and JavaScript, handling some backend operations with the Firebase database, and configuring the app to send push notifications to administrator users when new requests were created.
